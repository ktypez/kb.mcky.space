# DEC-001: Neobrutalist design with pure CSS

## Decision

Use pure CSS with neobrutalist styling (3px borders, hard offset shadows, bright accents) — no CSS framework.

## Rationale

- Lightweight — no Tailwind/postcss dependency
- Distinct visual identity
- Easy to theme via CSS custom properties

## Consequences

- All styling is hand-written CSS
- Responsive design must be managed manually
- Dark mode via `[data-theme="dark"]` attribute selector
