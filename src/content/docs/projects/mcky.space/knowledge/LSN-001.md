# LSN-001: Fixing android-arm64 native bindings for local dev

Switched syntax highlighter from shiki to Prism (zero native deps). Installed `@rolldown/binding-linux-arm64-gnu` directly (npm skips optional deps on android-arm64). Generated 22 missing `.mjs` ESM wrappers for unstorage (npm drops ESM files for dual CJS/ESM packages on this platform). Patched `os.networkInterfaces` in `astro.config.mjs` to return a fake `127.0.0.1` interface — Termux throws EACCES on `uv_interface_addresses`. Set `server: { host: '0.0.0.0' }` for LAN access. Use `--host 127.0.0.1` if the os.networkInterfaces patch causes issues.
