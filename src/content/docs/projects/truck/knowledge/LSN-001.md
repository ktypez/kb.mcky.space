# LSN-001: Reauthentication required before user update

## Lesson

Supabase requires explicit `signInWithPassword()` before `updateUser()` even when the user is already authenticated. Skipping this step causes auth errors.

## Resolution

Always call `signInWithPassword()` with stored credentials before `updateUser()`. The auth session alone is insufficient.
