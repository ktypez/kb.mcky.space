# DEC-015: Add notes field to client (`note`)

## Decision

Add a free-text `note` field ("บันทึกเพิ่มเติม") to clients for additional information,
editable on the add/edit forms and displayed on the detail page (admin + public `/c/[id]`).

## Changeset
- `app/types.ts`: `Client.note: string`
- `lib/db/schema.ts`: `note: text('note').notNull().default('')`
- `lib/db/migrations/0002_add_note.sql`: `ALTER TABLE clients ADD COLUMN note text NOT NULL DEFAULT ''`
- `lib/clients.ts`: `rowToClient`, `addClient`, `updateClient`, `validateClient` (note ≤5000 chars)
- `components/AddClientForm.tsx`: `note` state + `<textarea>` (max 5000) after address field
- `components/ClientDetail.tsx`: Notes `Card` below info card; action buttons moved to bottom;
  `NotePencil` icon import added

## Implementation notes
- `note` flows through `lib/storage.ts` / `lib/offline-db.ts` automatically (generic record storage)
- Displayed on both admin detail and public page (shared `ClientDetail` component)
- Action buttons (edit/delete, suggest/photo-request) were moved from top to bottom of the detail view
