# DOC-002: Trash item card display

# Trash item should display in card like main page

**File:** `src/components/TrashView.tsx`

Currently trash items render as simple list rows (muted background, text-only). They should use the same card component (`@/components/ui/card`) and layout pattern as `DesktopCardView` / `MobileCardList` on the main page — including photo thumbnail, shop name/name, deleted date, and action buttons (restore / force delete) pinned to the card footer.

- [x] Import `Card` from `@/components/ui/card`
- [x] Render each trash item as a `Card` matching `DesktopCardView` style (grid / flex layout)
- [x] Use `Client` type or extend `TrashItem` to include `images`, `badge` fields
- [x] Adapt `handleRestore` / `handleForceDelete` to card affordance
